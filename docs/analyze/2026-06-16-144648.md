## 핵심 패턴 (실제 관찰됨 — 문서 전용 아님)

- **상태**: Pinia(client tabs/activeRepo) + Vue Query(server). `useRepositoryQuery` 팩토리가 7 micro-hook 공유. `createGlobalState` **미사용 (0)**.
- **IPC**: `api/git.ts` 가 Rust command 와 1:1. `invokeWithTimeout` (30s 기본 / 5min long-running 자동 감지) + 네트워크 prefix 지수 백오프 재시도 + **NFC 정규화**(한글 안전).
- **에러**: Rust `AppError`(thiserror+serde) → `AppErrorPayload` 자동 직렬화 → `describeError`/`detectGitHint` 로 한글 힌트 → `useToast` (kind별, 1s dedup).
- **Rust 동시성/보안**:
  - `repo_mutation_guard()` per-repo TokioMutex (destructive IPC TOCTOU 직렬화) + `repo_lock()` lazy HashMap + `prune_idle_locks()` 무한증가 방지
  - `validate_repo_relative_path()` 4단계 (empty/`..` component-level/absolute/prefix) + `reject_dash_prefix()` (CWE-88) + `--end-of-options`
  - `git_run()` UTF-8/NFC/safe.directory `-c` 주입 + `GIT_SSH_COMMAND` per-profile 키 바인딩
  - `secret_mask` 13 regex 패턴
  - 로깅: `tracing::` 직접 호출 (구조화 필드), `#[instrument]` 매크로 **미사용**
- **컴포넌트**: `use*` composable + `useMutation`/`useQuery`. reka-ui 는 **BaseTooltip 1곳만** (대부분 native Vue + Tailwind).
