## 4. CI / 빌드 (GitHub Actions)

### 4-1. `.github/workflows/ci.yml`

```yaml
name: CI
on:
  push:
    branches: [main]
  pull_request:

jobs:
  test:
    runs-on: windows-latest
    steps:
      - uses: actions/checkout@v4
      - uses: oven-sh/setup-bun@v1
      - uses: dtolnay/rust-toolchain@stable
      - name: bun install
        run: bun install
        working-directory: apps/desktop
      - name: typecheck
        run: bun run typecheck
        working-directory: apps/desktop
      - name: lint (when ESLint v9 마이그레이션 완료)
        run: bun run lint
        working-directory: apps/desktop
      - name: cargo test
        run: cargo test --lib
        working-directory: apps/desktop/src-tauri
      - name: cargo clippy
        run: cargo clippy --all-targets -- -D warnings
        working-directory: apps/desktop/src-tauri
      - name: vitest (옵션)
        run: bunx vitest run
        working-directory: apps/desktop
        continue-on-error: true   # 15 plan Sprint 5 완료 전까지
```

### 4-2. `.github/workflows/release.yml`

```yaml
name: Release
on:
  push:
    tags: ['v*']

jobs:
  build:
    runs-on: windows-latest
    steps:
      - uses: actions/checkout@v4
      - uses: oven-sh/setup-bun@v1
      - uses: dtolnay/rust-toolchain@stable
      - run: bun install
        working-directory: apps/desktop
      - name: tauri build (signed)
        env:
          TAURI_SIGNING_PRIVATE_KEY: ${{ secrets.TAURI_SIGNING_PRIVATE_KEY }}
          TAURI_SIGNING_PRIVATE_KEY_PASSWORD: ${{ secrets.TAURI_SIGNING_PRIVATE_KEY_PASSWORD }}
        run: bun run tauri:build
        working-directory: apps/desktop
      - name: signtool (EV)
        env:
          EV_THUMBPRINT: ${{ secrets.EV_THUMBPRINT }}
        run: |
          signtool sign /sha1 $env:EV_THUMBPRINT /tr http://timestamp.sectigo.com /td sha256 /fd sha256 apps/desktop/src-tauri/target/release/bundle/msi/*.msi
      - name: Upload to release
        uses: softprops/action-gh-release@v2
        with:
          files: |
            apps/desktop/src-tauri/target/release/bundle/msi/*.msi
            apps/desktop/src-tauri/target/release/bundle/nsis/*.exe
            apps/desktop/src-tauri/target/release/latest.json
          generate_release_notes: true
```

### 4-3. `latest.json` 자동 생성

`tauri-plugin-updater` 가 빌드 시 `latest.json` 자동 생성. release asset 으로 업로드. 사용자 환경의 `tauri.conf.json` 의 endpoint 가 GitHub releases 의 `latest.json` URL 을 가리키도록.
