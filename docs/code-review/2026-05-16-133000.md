### ARCH-001 — Pattern 20 측정 lockstep drift (HIGH)

- **File**: `scripts/god-comp-check.mjs:15-39` + `scripts/re-verify.mjs:79-86`
- **Finding**: 본 sprint 의 vue3-composable-extraction skill Pattern 20 (script vs total LOC 분리) 가 정의됐으나 두 측정 도구 모두 **total LOC 만** 측정. Pattern 20 의 Layer A (script LOC) 분기 누락.
- **영향**: template-heavy 정상 컴포넌트 (script <200 + total ≥200, 예: c89 StatusTreeDirRow) 가 pre-commit hook 의 warning 으로 분류 → false positive. 신규 PR 의 template-heavy 추가가 부당하게 회귀로 표시됨.
- **권고 (load-bearing)**:
  - script tag offset 추출: `const m = src.match(/<script[^>]*>[\s\S]*?<\/script>/); const scriptLoc = m ? m[0].split('\n').length : 0`
  - 출력 분기: `Layer A 위반 (script ≥200): N건` + `Layer B-only (script <200, total ≥200): M건 — template SOT 통합 후보`
  - 환경변수 분리: `GOD_COMP_SCRIPT_THRESHOLD=200` + `GOD_COMP_TOTAL_THRESHOLD=400`
- **autoFixable**: Yes (script regex 추가만)
- **Convention violation**: Yes — Pattern 20 정의 ↔ 측정 SoT lockstep drift
