**현재 실 공격 경로 없음** (단일 origin Tauri desktop + XSS 부재 + WebView CSP). 단 외부 JSON 흡수 분기 추가 시 escalates.

**권고**:
```ts
for (const key of Object.keys(stored)) {
  if (key === '__proto__' || key === 'constructor' || key === 'prototype') continue
  // ...
}
```
